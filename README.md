## Getting Started

### Build

Build the application
* mvn
  ```sh
  mvn clean build
  ```
Build the docker image
* mvnw
  ```sh
  ./mvnw spring-boot:build-image
  ```

### Run

Start the application
* mvn
  ```sh
  mvn spring-boot:run
  ```

Navigate to [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)
